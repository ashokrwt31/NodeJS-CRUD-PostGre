# NodeJS-CRUD-PostGre

Node.js Authentication, Postgre and CRUD API with Email Verification.


# Register and Login User

Create a POST request to /api/userAuth/register
Create a POST request to /api/userAuth/login 

# CRUD operation for Employee details

POST request to create new Employee     /api/employee/
POST request to update Employee         /api/employee/:id
GET request to get all Employee         /api/employee/
GET request to get Employee by ID       /api/employee/:id
DELETE request to delete Employee by ID /api/employee/:id
